# Markdown写作指南

这篇文章将教你如何使用Markdown来写博客文章~ 📝

## 什么是Markdown？

Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的HTML文档。

## 基础语法

### 标题

使用`#`号来创建标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

- **粗体**：用`**文字**`
- *斜体*：用`*文字*`
- ~~删除线~~：用`~~文字~~`

### 链接和图片

```markdown
[链接文字](URL)
![图片描述](图片URL)
```

### 代码

行内代码用反引号：`code`

代码块用三个反引号：

```python
def hello():
    print("Hello, World!")
```

### 列表

无序列表：
- 项目1
- 项目2
- 项目3

有序列表：
1. 第一项
2. 第二项
3. 第三项

## 写作建议

1. **保持简洁** - 一个段落只讲一件事
2. **善用标题** - 帮助读者快速浏览
3. **代码高亮** - 指定语言获得更好的显示效果
4. **适当留白** - 让文章更易阅读
