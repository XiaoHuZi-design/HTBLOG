# 欢迎来到机械仔的博客！

你好呀！欢迎来到我的小站~ ✨

## 关于这个博客

这是一个**二次元风格**的个人博客系统，主要用来记录我的学习笔记、技术分享和生活随想。

### 主要功能

- 📝 **Markdown文章** - 支持完整的Markdown语法
- 💾 **GitHub存储** - 文章自动保存到GitHub仓库
- 💬 **Gitalk评论** - 基于GitHub Issues的评论系统
- 🌸 **二次元风格** - 樱花飘落、看板娘陪伴
- 🌙 **日夜主题** - 支持明暗主题切换

## 代码示例

这里是一段JavaScript代码：

```javascript
// 二次元打招呼
function sayHello() {
    const greetings = [
        "欧尼酱，今天也要元气满满哦~",
        "你好呀！今天也是可爱的一天！",
        "(*^▽^*) 嘿嘿~"
    ];
    const random = Math.floor(Math.random() * greetings.length);
    console.log(greetings[random]);
}

sayHello();
```
